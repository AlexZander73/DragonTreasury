# Treasure Sprite Set

This folder now includes a first-pass stylized sprite set used at runtime:

- `coin.svg`
- `gem.svg`
- `artifact.svg`
- `legendary-relic.svg`
- `cursed-item.svg`
- `metal-idol.svg`
- `arcane-crystal.svg`
- `scroll-capsule.svg`

`src/scene/TreasureVisuals.ts` uses these sprites first and keeps vector fallback geometry beneath them.
